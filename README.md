# C-simple-calculator
A lightweight and efficient command-line calculator written in **C**. This application performs basic arithmetic operations based on user input.

## Features
* **Addition (+)**: Adds two floating-point numbers.
* **Subtraction (-)**: Subtracts the second number from the first.
* **Multiplication (*)**: Multiplies two numbers.
* **Division (/)**: Divides the first number by the second with a **zero-division check**.
* **Error Handling**: Validates operators and prevents mathematical errors.
